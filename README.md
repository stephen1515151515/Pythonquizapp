# 🧠 AI Quiz Generator
### Internship Mini Project — Think Champ PV Ltd

---

## 📌 Project Overview
An AI Quiz Generator built in **Python** (terminal) and also as a **Web App** (HTML/CSS/JS).
It asks multiple-choice questions, validates answers, tracks scores, and saves results.

---

## 📁 Project Structure
```
AI_Quiz_Generator/
├── quiz.py          ← Python terminal quiz (main project)
├── index.html       ← Beautiful web-based version
├── questions.txt    ← Question bank (reference)
├── scores.txt       ← Auto-saved score log
└── README.md        ← This file
```

---

## 🚀 How to Run

### ▶ Web Version (Recommended — looks amazing!)
Just double-click **`index.html`** to open in your browser. No installation needed!

### ▶ Python Terminal Version
```bash
# Step 1: Install colorama (for colors)
pip install colorama

# Step 2: Run the quiz
python quiz.py
```
